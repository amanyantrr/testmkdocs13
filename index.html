Vayu 3GH/3GHP Cape

Descriptions

This M2M cape provides seamless highspeed internet connectivity to the Beaglebone via 3G/UMTS network anytime anywhere. It also has an optional GPS add-on for location and time synchronization (1pps signal, 2 microsecond jitter accuracy).

VAYU-3GH Cape is catering to increasing data throughput requirements in M2M sector, specifically for video, voice-over-IP, and high speed realtime logging applications. It is based on industry standard Qualcomm MSM6290 Chipset.

VAYU-3GH Cape is backward compatible, and supports GSM/GPRS/WCDMA/ HSUPA network. It supports WCDMA 2100/850 and WCDMA 2100/900 bands and standard Quad-band (850/900/1800/1900) for 2G networks. Max DL speed is 7.2 Mbps, max UL speed is 5.76 Mbps.

VAYU-3GHP Cape provides an add-on GPS module integrated with WCDMA modem which provides standard compliant NMEA based geolocation and global time synchronization (1pps, 0.2 microsecond jitter accuracy).
